Understanding CSS Units

A guide to Absolute (px) and Relative (%, em, rem, vh, vw) units.

1. Absolute Units (px)

Pixels are fixed. They do not change based on user settings.

300px Wide Box

2. Percentage (%)

Relative to the width of the parent container.

50% of Parent

3. Typography Units (em vs rem)

REM: Always looks at the Root (HTML) font size.
EM: Looks at its immediate parent's font size.

4. Viewport Units (vh / vw)

Relative to the actual browser window size.

50vw (Half the screen width)
20vh (20% of screen height)
